h1. Radiant CMS Filmstrip Extension


h1. Features

* Allows you to create a filmstrip gallery in Radiant and display on a page.
* Loading order of images in the filmstrip can be adjusted using drag-and-drop reordering.
* Filmstrip images are pulled-in to the Flash via and XML file produced in the *.rxml views in app/views/filmstrips.

h1. Requirements

<p>*SWFObject*</br />
swfobject, "http://blog.deconcept.com/swfobject":http://blog.deconcept.com/swfobject</p>

<p>*Gems/Plugins*<br /> 
paperclip, "http://github.com/thoughtbot/paperclip/tree/master":http://github.com/thoughtbot/paperclip/tree/master</p>

h1. How to Install

Grab the extensions from GitHub and place it in vendor/extensions
<pre><code>
  git clone git@github.com:scullygroup/radiant-filmstrip.git vendor/extensions/filmstrip
</code></pre>

Run the following commands:
<pre><code>
  rake radiant:extenstions:filmstrip:migrate
  rake radiant:extenstions:filmstrip:update
</code></pre>

h1. How to Use

The filmstrip is pulled into Flash using a RESTful URL that provides the XML like so:
<pre><code>/filmstrip/:slug</code></pre>

<p>Where <code>:slug</code> is the page slug of the current Radiant Page.</p>

<p>In your template, create a section like the following:
  <pre><code>
    <div id="slideshow"></div>

    	<script type="text/javascript" charset="utf-8">
    	var flashvars = {
    	    xml_path:"/filmstrip/<r:slug/>"
    	    };
      var params = {
          menu: "false",
          wmode: "transparent"
          };
      var attributes = {};

      swfobject.embedSWF("swf/slideshow.swf", "slideshow", "100%", "290","9.0.0", "", flashvars, params, attributes);
      </script>
  </code></pre>
  
  <p>The code above uses SWFObject to create the Flash embed code, using XML to populate the filmstrip.  This pulls all images for the current page.</p>
